# autoHeal

A Windows desktop productivity companion that integrates with **Notion** to automate daily routines.

- Auto-joins upcoming meetings from Notion calendar events  
- Context-aware auto-scrolling on YouTube / Instagram Reels  
- Scheduled task execution (e.g., send messages/notifications)  
- Workflow switching (close apps after meetings, reopen IDEs like Visual Studio)  
- System tray application with minimal UI

Built as part of a sponsored 5-week project using **Qt 6.8.3 LTS** on Windows.
